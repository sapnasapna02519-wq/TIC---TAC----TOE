# Tic Tac Toe
This is a simple **Tic Tac Toe game**  
built using **HTML, CSS, and JavaScript**.  
Two players (X and O) can play against  
each other and enjoy interactive gameplay.  

---
## Features
- Interactive gameplay with two players (X amd Y)  
- Turn-based system  
- Reset button to start a new game  
- Simple and responsive design  

---
## How to play
1.Player X starts the game.  
2.Players take turns placing X or O in empty calls.  
3.The first player to align 3 marks  
horizontally,vertically,or diagonally wins.  
4.If all cells are filled and no one wins,the game ends in a draw.  
5.Click the **Reset** button to start a new game.
